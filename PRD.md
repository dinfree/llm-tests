프로그램을 개선하려고 해. 다음과 같은 요구사항을 반영해서 프로그램을 다시 만들어줘.

# 프로그램 시나리오
1. 모델명을 .env 에서 읽지 말고 프로그램 시작시 서버에 쿼리 보내서 모델명을 가지고 와서 사용해.
2. 프로그램은 모두 삭제하고 다음 요청에 따라 다시 만들어.
3. 프로그램 시작시 메뉴를 띄우는데 앞에서 가져온 모델명과 다음 메뉴를 보여줘 "1.단일테스트, 2.다중테스트, 3. 결과보기, 4.종료"
4. 단일테스트는 현재 token_test.py 의 코드 내용을 참조해서 다시 구현.
5. 다중 테스트는 동시요청수를 추가로 입력받도록해. 입력 형태는 10, 100, 200... 형태야.
6. 입력이 완료되면 시작한다는 메시지와 함께 진행 상황을 로그로 남겨
7. 완료되면 출력 형태에 따라 결과를 보여주고 결과는 sqlitedb에 저장해. 출력 정보는 다음과 같아.
    - Total Tokens, Total Duration, First Token Latency(TTFT), Throughput(TPS)
    - 다중 테스트의 경우에는 동시요청수도 같이 보여줘.
8. 실험결과 차트는 matplotlib 라이브러리를 사용해서 그려줘. 차트 텍스느는 모두 영어로 해줘.
9. 차트 이미지는 result 폴더에 저장해줘. 이미지 파일명은 모델명_테스트유형_동시요청수.png 형태로 저장해줘. 예를 들어, gpt-3.5_single_10.png 이런식으로 저장해줘.
10. 결과보기를 선택했을때 DB 내용을 보기 좋게 텍스트 테이블 형태로 보여줘.

# 요구사항
- 프로그램 시나리오에 따라 프로그램을 다시 만들어줘.
- 설명이 부족한 부분은 자유롭게 제안하도록.
- 현재 결과 이미지는 prev_result 폴더로 옯겨.
- deprecated된 라이브러리는 사용하지 말고 최신 라이브러리를 사용해.

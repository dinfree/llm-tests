# LLM 테스트 프로그램
## 개요
- 자체 운영중인 프록시 서버를 통해 제공되는 모델들에 대한 테스트 프로그램입니다.
- 서버에서 제공되는 모델은 
- 서버에서 제공되는 API 는 Open AI API 와 호환됩니다.

## 프로그램 구성
### 메뉴
- 1.text 모델 테스트, 2.image 모델 테스트, 3.embedding 모델 테스트, 4.종료

### 테스트 절차
1. 메뉴에서 모델 유형 선택
2. text 모델 테스트
- 사용자 입력: 프롬프트 입력
- 시스템 응답: 모델에서 생성된 텍스트 출력
3. vision 모델 테스트
- 사용자 입력: sample.jpg 파일 자동 업로드, 프롬프트는 입력
- 시스템 응답: 이미지 내용에 대한 프롬프트 요청결과 출력
4. embedding 모델 테스트
- RAG 방식
- 파일첨부: 파일명을 직업 입력(예: sample.txt, sample.pdf)
- 사용자 입력: 프롬프트는 입력
- 시스템 응답: 벡터 임베딩 결과를 사용해 입력에 맞는 답변을 출력
5. 출력 포맷
- 시스템 응답을 텍스트 stream 형태로 출력
- 응답 종료시 입력토큰 수, 출력토큰 수, TTFT, TPS 포함 해서 출력.

### 기술 스택
- Python 3.14.x, 현재 .venv 환경에서 실행
- .env 파일에서 API_KEY, BASE_URL 가져와서 사용
- Langchain 라이브러리 활용
- OpenAI API 호환 프록시 서버와 통신
- 파일 업로드 및 처리 기능 포함
- 텍스트, 이미지, 임베딩 모델 테스트 기능 구현
- 벡터DB는 faiss vector DB 사용
